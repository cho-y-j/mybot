       ├ 경쟁자 동향
       └ 오늘 추천 행동 3가지

```

             → 이상 시에만 즉시 텔레그램 알림
             → 이상 없으면 패스 (서버 부하 없음)

### 오후 (자동)
```
13:00  수집
14:00  텔레그램 오후 브리핑
       ├ 오전 대비 변화
       ├ 실시간 이슈
       └ 오후 대응/콘텐츠 제안
```

### 마감 (자동)
```
17:00  수집
18:00  텔레그램 일일 보고서
       ├ 오늘 종합 (수치)
       ├ 경쟁자 대비 우리 위치
       ├ 부족한 점 체크리스트

```

콘텐츠 도구 → 블로그 주제/태그 생성 (검색량 기반)
```

## 2. 데이터 수집 (핵심!)

### 수집 소스 우선순위

| 순위 | 소스 | 수집 내용 | 여론 가치 |
|------|------|----------|----------|
| 1 | **네이버 뉴스 + 댓글** | 기사 제목/본문/감성 + 댓글 감성/반응 | ★★★★★ |
| 2 | **맘카페/지역카페** | 학부모/지역민 실제 대화 | ★★★★★ |
| 3 | **유튜브 + 쇼츠 + 댓글** | 영상 노출 + 시청자 반응 | ★★★★ |
| 4 | **네이버 블로그** | 장문 의견/리뷰 | ★★★ |
| 5 | **검색어 트렌드** | 유권자 관심사 변화 | ★★★★ |
| 6 | **SNS (인스타/X)** | 확산력/해시태그 | ★★★ |
| 7 | **여론조사** | 정량적 지지율 | ★★★★★ |

### 수집별 구현 방법

```
네이버 뉴스 댓글:
  → 뉴스 URL에서 댓글 API 호출
  → 댓글 수, 공감/비공감, 텍스트 수집
  → AI 감성 분석

맘카페/지역카페:
  → 네이버 카페 검색 API (이미 키 있음)
  → 키워드: "교육감", "학교급식", 후보이름 등
  → 자동 생성된 키워드로 검색

유튜브 쇼츠:
  → YouTube API type=video, videoDuration=short
  → 조회수/좋아요/댓글 수집

검색어:
  → 네이버 DataLab (이미 작동)
  → 네이버 검색광고 API (이미 작동)
```

## 3. AI 분석 (핵심!)

### AI가 하는 일

```
1. 감성 분석 — 뉴스/댓글/커뮤니티 긍정/부정/중립
   → Claude CLI로 맥락 이해 (폭력 방지 = 긍정)
   → 확실하지 않으면 중립

2. 경쟁자 대비 분석 — 우리가 부족한 점 자동 체크
   → "검색량 13% — 인지도 심각 부족"
   → "맘카페 언급 0건 — 학부모 접점 없음"
   → "경쟁자 A는 AI교육 선점 — 차별화 필요"

3. 전략 추천 — 구체적 행동 제안
   → "이번 주 블로그: '충북 돌봄 현황' (검색량 1,350)"
   → "유튜브 쇼츠: 3분 정책 설명 시리즈"
   → "맘카페 키워드 '학교급식' 관련 글 작성"

4. 위기 감지 — 부정 뉴스/여론 급변
   → 즉시 알림 + 대응 방안 제안

5. 콘텐츠 생성 도움
   → 블로그 초안 (AI)
   → 해시태그 (검색량 기반)
   → SNS 문구 (선거법 체크 포함)
```

## 4. 대시보드 (실시간 조회)

### 메인 대시보드
```
┌──────────────────────────────────────────┐
│ D-61 | 뉴스 37건 | 긍정률 43% | 위기 1건  │ ← KPI
├──────────────────────────────────────────┤
│ [AI 인사이트]                              │
│ "윤건영 대비 노출 58% — 이번 주 공약 발표  │
│  필요. 맘카페 접점이 없어 학부모 관련      │
│  콘텐츠 우선 추천"                         │
├──────────────────────────────────────────┤
│ [경쟁자 대비 체크]                          │
│  ⚠ 검색량: 경쟁자 38% 수준                 │
│  ⚠ 맘카페: 경쟁자만 언급                   │
│  ✅ 긍정률: 1위                            │
│  ⚠ 유튜브: 콘텐츠 부족                     │
├──────────────────────────────────────────┤
│ [최근 뉴스] 시간순, 링크+감성+요약          │
│ [여론 동향] 커뮤니티/댓글 요약              │
│ [검색 트렌드] 후보별+이슈별                 │
└──────────────────────────────────────────┘
```

### 하위 페이지들
```
뉴스 분석 — 실데이터, 필터, 링크, AI 요약
여론 동향 — 맘카페/커뮤니티/댓글 분석
검색 트렌드 — 실시간 DataLab + 키워드 검색량
후보 비교 — 경쟁자 대비 부족한 점 중심
유튜브 — 영상+쇼츠+댓글+링크
여론조사 — 질문별 결과 + 추이
과거 선거 — AI 심층 분석 (선거 패턴)
콘텐츠 도구 — 검색량 기반 주제/태그 + AI 초안
AI 챗 — 실데이터 기반 대화
보고서 — 자동 생성 기록
```

## 8. 구현 순서

### Phase A: 데이터 수집 강화 (기초)
```
A1. 뉴스 댓글 수집 추가
A2. 맘카페/지역카페 수집
A3. 유튜브 쇼츠 + 댓글 수집  
A4. 블로그 본문 수집 강화
A5. 실시간 위기 감지 (30분 체크)
```

### Phase B: AI 분석 고도화
```
B1. Claude CLI로 뉴스/댓글 감성 분석
B2. 경쟁자 대비 부족점 자동 분석
B3. AI 전략 추천 (구체적 행동)
B4. AI 보고서 자동 생성 (오전/오후/마감)
B5. AI 콘텐츠 초안 생성
```

### Phase C: 대시보드/UI 재설계
```
C1. 메인 대시보드 — AI 인사이트 + 경쟁자 체크 중심
C2. 뉴스 분석 — 댓글 포함, 시간순, 링크
C3. 여론 동향 — 커뮤니티/댓글 통합
C4. 검색 트렌드 — 실시간 + 키워드 검색량
C5. 후보 비교 — 부족한 점 중심
C6. 유튜브 — 쇼츠+댓글+링크
C7. 여론조사 — 질문별 + 에러 수정
C8. 콘텐츠 도구 — 검색량 연동 + AI 초안
C9. 보고서 — AI 자동 생성 보기
```

### Phase D: 보고서/텔레그램
```
D1. 오전 브리핑 AI 자동 생성 + 발송
D2. 오후 브리핑
D3. 일일 보고서
D4. 위기 즉시 알림
```

### Phase E: 완성도
```
E1. 관리자 패널 완성
E2. 온보딩 → 자동 수집 시작까지 원스텝
E3. 전체 통합 테스트
E4. 모바일 반응형
```
